Overview:

This module focuses on software management in Ubuntu using package managers. Learners are introduced to tools such as apt for installing, updating, and removing software packages. It also compares terminal-based installation with graphical methods like the Ubuntu Software Center, helping users understand different approaches to managing applications.

Learning Objectives:

  • apt update and upgrade
  • Installing apps (apt install)
  • Removing Apps
  • Software center vs Terminal
