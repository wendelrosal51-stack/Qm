Overview:

This module introduces learners to the fundamentals of Linux, focusing on Ubuntu as the primary platform. It explains the concept of operating systems, highlights the differences between Linux and other systems (e.g., Windows), and emphasizes the importance of Linux in real-world applications such as servers and development environments.

Learning Objectives:

  • How to use terminal
  • How to utilize Recovery Mode
  • What are the basic commands the Ubuntu system?
  • How to update applications
  • System Maintenance
